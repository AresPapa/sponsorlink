# Sponsor Overview Page notes

## 1. High-level stats ("at a glance")

- Active Sponsorships
- Pending Negotiations
- Total Proposals Sent
- Proposals Accepted
- Money Spent (total)
- Average Budget per Sponsorship
- Upcoming Payments / Pending Invoices

## 2. Current Sponsorships

- Event name
- Organizer
- Tier
- Status
- End date
- Quick Action (View details)

## 3. Active Negotiations

- Event name
- Your latest offer
- Their latest counter
- Negotiation status (in progress / awaiting response)
- Quick action (open negotiation)

## 4. Recent Activity

- Last proposals sent
- Last messages exchanged in negotiations
- Payments made
- Sponsorships accepted or rejected
- Any updates from ongoing events

## 5. Quick Actions

- Start New Sponsorship (search events button)
- View all events
- Open Wallet / Payment Info
- Profile Settings

## 6. Recommended Events

- Events matching their preffered categories
- Events that fir their budget range
- Events trending or newly added

## 7. Budget Snapshot

- Remaining monthly budget
- Upcoming expected expences
- Optional: "Spending vs Budget" mini-graph (very small, sublte)

## 8. Messages / Notifications

- Unread negotiation messages
- System notifications
- Payment reminders
- Proposal updates

## 9. Profile Summary (mini version)

- Sponsor name
- Logo
- Industry
- Budget Range
- Primary contact
